## ➕ Cara Menambahkan Repository Cloudstream

Pilih salah satu metode di bawah ini untuk memasang repositori **MiraiExt**:

### 1. Instalasi Langsung (Direct Install)
<a href="cloudstreamrepo://raw.githubusercontent.com/arranoust/MiraiExt/builds/repo.json">
<img src="https://img.shields.io/badge/Direct_Install-CloudStream-005FFF?style=for-the-badge" alt="Direct Install">
</a>

### 2. Menggunakan Shortcode 
1. Buka aplikasi **CloudStream**.
2. Masuk ke **Setelan** > **Ekstensi**.
3. Klik **Tambah Repositori**.
4. Ketik `mirai` pada kolom kedua (URL Repositori atau Kode Pendek).
5. Klik **Tambah Repositori**.

### 3. Instalasi Manual
Jika cara di atas gagal, kamu bisa menambahkannya secara manual:
1. Salin URL repositori berikut:
   ```text
   https://raw.githubusercontent.com/arranoust/MiraiExt/builds/repo.json
   ```
2. Buka aplikasi **CloudStream**.
3. Masuk ke **Setelan** > **Ekstensi**.
4. Klik **Tambah Repositori**.
5. Paste URL repositori pada kolom kedua (URL Repositori atau Kode Pendek).
6. Klik **Tambah Repositori**.
